Det största problem jag stötte på i Javascript var att helt enkelt få koden att fungera som jag ville. Det var helt enkelt svårt att jobba med ett nytt språk och förstå det eftersom vi inte gått igenom det så mycket. Jag fixade det till slut när jag blev mer bekant med språket.

Jag gjorde inte så mycket i Javascript förutom min slideshow eftersom jag inte förstår själva språket helt och hållet än. Jag ville heller inte bara kopiera in t.ex en shopping cart in i min hemsida eftersom jag inte skulle förstå hur koden fungera med min exsisterande kod.

Problemen jag hade i HTML var relativt enkla. Att namge divs & att hitta rätt namn hade jag ett par problem med men det är inte så konstigt eftersom jag inte har jobbat med HTML speciellt länge. Jag hade också lite problem med länkar samt bilder men de större problemen hände i CSS efter jag hade kodat klart HTML.

Sammanfattningsvis så hade jag inte några större problem i HTML och de flesta lösningar säger sig själv. Jag behöver bara lite mer erfarenhet med språket för att få bort slarv fel.

Jag skulle säga att jag hade mest problem med CSS eftersom det är extremt många olika variabler & kommandon. Jag tyckte även det var svårt med flexbox men det lärde jag mig över tid.

Men ja, jag skulle säga att lära sig när man ska använda vad är en process & även nu när jag är klar men projektet kan det fortfarande vara svårt att veta vad man behöver göra ibland.
